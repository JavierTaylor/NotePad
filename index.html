NotePad

I made the NotePad application to get to know Java, XML, Android SDK, and mobile development in general. It is a text saving application for Android devices. It is simple, easy to use, and is integrated with your other applications so that you can share your text by Gmail, Facebook, and Twitter, to name but a few. The code for it can be downloaded off of GitHub at https://github.com/JavierTaylor/NotePad/releases (which includes zip, tar, and an apk to install on your phone).
